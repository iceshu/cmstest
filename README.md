## 目录结构

bff //api 接口

admin //react antdpro 后台

webts // nuxt 前台展示

## 技术栈

ts nuxt midwayjs react antd

## 开始

1、分别进入目录安装依赖

2、还原数据 --- 在 bff 目录 执行 node ./build/restore.js

3、

- bff ---npm run dev 开发
- admin ---npm start 开发
- webts ---npm run dev 开发
